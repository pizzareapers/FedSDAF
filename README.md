# Federated Domain Generalization with Adapters
﻿
A framework for federated domain generalization using adapter-based fine-tuning of Vision Transformers (ViT). This approach enables efficient knowledge sharing across multiple domains while preserving domain-specific knowledge.
﻿
## Requirements
﻿

﻿
